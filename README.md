# NAME
prove-with-sound - Execute prove with sound


# DESCRIPTIONS
prove-with-sound executes prove with sound.  
If test is passed, it playbacks the sound of success. Else, it plays sound of failure.


# USAGE
    $ prove-with-sound target_code
